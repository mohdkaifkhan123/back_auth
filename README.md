Register API
http://localhost:8080/register
In body
For Example
{
    "username":"shyam",
    "password":"s@123"
}

Login API
http://localhost:8080/login
In body
For Example
{
    "username":"shyam",
    "password":"s@123"
}

For to test Api if authentication working or not
http://localhost:8080/verify(add token in Authorization--->Bearer-->Bearer Token)
